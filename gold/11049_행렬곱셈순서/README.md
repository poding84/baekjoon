# 행렬 곱셈 순서

## 사용 알고리즘 : dp

### 문제 설명

행렬들이 주어지고 해당 행렬들을 곱셈할 때 가장 최소한으로 필요한 연산의 숫자를 출력하는 문제.

### 풀이 설명

dp[i][j] : i번째 행렬부터 j번째 행렬까지 곱셈할 때 최손하으로 필요한 연산의 숫자.

점화식은 다음과 같다.

dp[i][j] = min(dp[i][k] + dp[k][j] + mat[i] * mat[k] * mat[j]), k는 i+1부터 j-1까지.

### 시간복잡도 : N^2

최악의 경우 dp배열을 모두 채워야 하므로, N^2
