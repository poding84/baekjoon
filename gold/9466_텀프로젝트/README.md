# 텀 프로젝트

## 사용 알고리즘 : 위상정렬

### 문제 설명

i 학생은 a[i] 학생과 같은조가 되고싶어 한다.

같은 조가 되기 위해서는 선택한 학생들끼리 사이클을 이뤄야 한다.

조를 이루지 못한 학생 수 출력하기.


### 풀이 설명

모든 학생의 진입차수를 계산한다. 진입차수가 0인 학생들은 사이클을 이루지 못한것이므로 개수를 기록해둔다.

그리고 나서, 해당 학생들을 제거하고 해당 학생들이 가리키고 있었던 학생들의 진입차수를 1 감소한다.

위의 과정을 큐가 빌 때까지 반복한다.


### 시간복잡도 : N

학생수만큼 반복문을 돌면된다.

### 어려웠던 점

쉽다.
